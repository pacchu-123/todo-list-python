# Python Console To-Do List Application

This is a simple console-based To-Do List application developed 
using Python. The program allows users to manage their daily 
tasks easily from the terminal.

## Features
- Add new tasks
- View all tasks
- Remove completed tasks
- Tasks saved permanently using a text file

## Technologies Used
- Python
- File Handling
- Lists and String Manipulation

## How to Run
1. Download the project files
2. Open terminal inside the folder
3. Run:
   python todo.py

## Purpose
This project was created as part of a Python Developer Internship 
task to demonstrate understanding of Python basics, file handling, 
and CLI application development.
